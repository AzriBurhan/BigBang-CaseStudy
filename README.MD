BigBang Case Study
Description

This script generates numbers from 1 to 100 and replaces values based on the following rules:

Multiples of 3 → BIG

Multiples of 5 → BANG

Multiples of 3 and 5 → BIGBANG

Others → Number as string

The results are saved into output.json.

Instructions
1. Open a new terminal

Go to the project folder:

cd BigBang-CaseStudy
2. Run the script
node bigbang.js
3. Check the output

After running, a file named:

output.json

will be created automatically in the same folder.
